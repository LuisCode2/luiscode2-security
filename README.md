# luiscode2 - Servicios de Seguridad Informática

## 🔐 Descripción

luiscode2 es una página web profesional para publicitar servicios de hacking ético y ciberseguridad. Ofrece:

- **Pruebas de Penetración**: Identificación de vulnerabilidades en infraestructura
- **Auditorías de Seguridad**: Evaluación completa de postura de seguridad
- **Capacitación**: Formación en seguridad ofensiva y defensiva

## ✨ Características

- Diseño moderno y minimalista
- Interfaz responsiva (móvil, tablet, desktop)
- Navegación fluida con scroll suave
- Secciones: Inicio, Servicios, Portafolio, Contacto
- Formulario de contacto funcional
- Animaciones elegantes
- Tema oscuro profesional

## 🛠️ Tecnologías

- HTML5
- CSS3 (Grid, Flexbox, Animaciones)
- JavaScript Vanilla
- Diseño responsive

## 📁 Estructura del Proyecto

```
luiscode2-security/
├── index.html      # Página principal
├── styles.css      # Estilos y tema
├── script.js       # Interactividad
```

## 🚀 Cómo Usar

1. Clona o descarga el repositorio
2. Abre `index.html` en tu navegador
3. Personaliza el contenido con tu información
4. Despliega en un servicio de hosting (GitHub Pages, Netlify, Vercel, etc.)

## 🎨 Personalización

### Cambiar Colores
Modifica las variables CSS en `styles.css`:

```css
:root {
    --primary-color: #0f172a;      /* Azul oscuro */
    --accent-color: #3b82f6;       /* Azul brillante */
    /* ... más variables */
}
```

### Actualizar Información de Contacto
Edita la sección de contacto en `index.html`:

```html
<p><strong>Email:</strong> tu-email@luiscode2.com</p>
<p><strong>Teléfono:</strong> +1 (555) 123-4567</p>
```

### Agregar Casos de Éxito
Añade nuevos `portfolio-item` en la sección de portafolio:

```html
<div class="portfolio-item">
    <h4>Tu Proyecto</h4>
    <p>Descripción...</p>
    <span class="tag">Pentesting</span>
</div>
```

## 📧 Formulario de Contacto

Actualmente, el formulario muestra un mensaje de confirmación. Para hacerlo funcional:

1. **Backend Node.js**: Configura un servidor express
2. **Servicio Email**: Usa Nodemailer, SendGrid, Mailgun, etc.
3. **Alternativa**: Usa servicios como Formspree, Basin, o similar
